# üìå Project Deep Dive

1) Restaurant Bill Analysis (Tips Dataset)
   - Objective: Analyze tipping behavior (gender, time, day).
   - Insights: Who tips more? Which day has most revenue?

2) Supermarket Sales Analysis
   - Objective: Explore product performance and daily revenue trends.
   - Insights: Top-selling products, average quantity sold.

3) Students Marks Analysis
   - Objective: Study student performance & grade distribution.
   - Insights: Top scorers, class averages, categorized grades.

4) Employee Data Analysis
   - Objective: Basic HR analytics.
   - Insights: Highest salary, department averages, youngest/oldest employee.

5) Movies Dataset Analysis
   - Objective: Analyze movie ratings, genres, revenue.
   - Insights: Popular genres, top-rated films, correlation checks.
